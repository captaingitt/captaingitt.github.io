Основные команды Git для Windows

  • Создание Git-репозитория: $ git init
  • Игнорирование файлов: $ cat .gitignore
  • Определение состояния файлов: $ git status
  • Коммит изменений: $ git commit
  • Удаление файлов: $ git rm filename
  • Перемещение файлов: $ git mv file_from file_to
  • Отслеживание новых файлов: $ git add filename или $ git add .
  • Операции отмены: $ git commit --amend
  • Отмена изменений в файле: $ git checkout -- filename
